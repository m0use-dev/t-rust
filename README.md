## 環境構築

### インストール
```
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
export PATH="$HOME/.cargo/bin:$PATH"
```

### バージョン確認
```
rustc -V
cargo -V
rustup -V
```

### コンパイル
```
rustc index.rs
```

### 実行
```
./index
```
